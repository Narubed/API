# wallet_history

## https://api.nbadigitalservice.com/v1/platform

### การดึงข้อมูลของ wallet_history ทั้งหมด

```http
GET /wallet_history
```

### การดึงข้อมูลของ wallet_history ผ่านไอดี

```http
GET /wallet_history/:_id
_id=62860054821f7b69619f254
```

```http
GET /wallet_history/member/:_id
_id=62860054821f7b69619f254
```
